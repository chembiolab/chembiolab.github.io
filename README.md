# chembiolab.github.io

This is the website for the Drug Target and Biomarker Discovery Laboratory, built with [Quarto](https://quarto.org), an open-source scientific and technical publishing system.

## Deployment

- Use `quarto render` to build the site locally into `docs/`.
- Use `quarto preview` to preview changes before publishing.
- Push the rendered `docs/` folder to `main`; under repo settings → **Pages**, the source should be set to the `docs/` folder on the `main` branch.
